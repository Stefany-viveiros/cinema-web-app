# 🎬 Cinema Web App

Plataforma web de cinema desenvolvida com foco em experiência do usuário, consumo de API e simulação de funcionalidades reais de um sistema moderno de exibição de filmes.

---

## 🚀 Sobre o Projeto

O **Cinema Web App** foi desenvolvido com o objetivo de simular a experiência de um usuário dentro de um ambiente de cinema digital, integrando funcionalidades reais como listagem de filmes, seleção de assentos e pedidos de bomboniere sem a necessidade de sair da sessão.

O projeto também explora conceitos importantes de desenvolvimento web, como consumo de APIs, organização de código e construção de interfaces interativas.

---

## 🎥 Funcionalidades

* 📌 Listagem de filmes em cartaz (consumo de API TMDB)
* 🍿 Seção de estreias e filmes em destaque
* 🎬 Banner dinâmico com destaque de filmes
* 🪑 Seleção de assentos com simulação de ocupação
* 🛒 Sistema de bomboniere com carrinho de compras
* 🎟 Simulação de compra de ingressos
* 🎬 Modo sessão (experiência imersiva)

---

## 🧠 Diferenciais

* Foco em **experiência do usuário (UX)**

## ▶️ Como Executar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/Stefany-viveiros/cinema-web-app.git
```

2. Acesse a pasta do projeto:

```bash
cd cinema-web-app
```

3. Abra o arquivo `index.html` no navegador

---

## 🔑 Configuração da API

1. Crie uma conta no TMDB
2. Gere sua API Key
3. No arquivo `js/api.js`, substitua:

```javascript
const API_KEY = 'SUA_API_KEY_AQUI';
```
